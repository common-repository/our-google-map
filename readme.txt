== Description ==

This plugin helps Show google map by shortcode. It accept three paramiter place,height,width and zoom.
Use: [golema place="type which place are show" height="Use Number" width="Use Number" zoom="Use Number"]

Deafult: place= "Mirpur 11"
         height = "600",
         width  ="500",
         zoom   = "13"
